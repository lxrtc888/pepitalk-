# Feature List

## Core Features (v1)
1. Mobile crowd exploration with bi-directional moving characters
2. Tap a character to trigger approach animation and chat mode
3. Character personas with distinct identity and style
4. Persistent user-character memory between sessions
5. Relationship state progression based on interaction history
6. Safety and fallback handling for sensitive prompts or failures

## Important Features (v2)
1. Voice output for characters
2. Emotional micro-animations and richer character motions
3. Character-to-character ambient interactions in crowd scene

## Future Iterations
1. Episodic story arcs and scheduled events
2. Content operation tools for adding/updating characters
3. Analytics dashboard for retention and engagement optimization
