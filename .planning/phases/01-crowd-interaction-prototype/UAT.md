# Phase 1 UAT Checklist

## Device and Display
- [ ] Opens correctly on iOS Safari (recent version)
- [ ] Opens correctly on Android Chrome (recent version)
- [ ] Layout is readable on 360x800 and 390x844 viewports

## Crowd Experience
- [ ] Crowd appears within 3 seconds on good network
- [ ] Characters move in both left and right directions
- [ ] Movement appears stable and non-jittery for 60 seconds

## Interaction
- [ ] Tapping a moving character consistently selects it
- [ ] Selected character visibly approaches before chat opens
- [ ] Chat can be closed and user returns to live crowd state

## Quality
- [ ] No blocking error shown to user during basic flow
- [ ] Input remains responsive while crowd is moving
- [ ] No obvious state desync between selected character and chat identity
