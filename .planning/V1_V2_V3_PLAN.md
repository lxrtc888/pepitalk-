# V1-V3 Delivery Plan

## V1 (Now): Playable Mobile Core Loop

### Goal
Validate the loop: discover crowd -> tap character -> chat -> local memory continuity.

### Deliverables
- Mobile crowd movement and tap selection
- Character approach and chat panel
- Local memory persistence on device
- Memory retrieval before every response turn
- Basic memory keyword search UI

### Exit Criteria
- Users can complete the loop in under 15 seconds from page load.
- Returning user sees memory-aware opening line.

## V2: Character Depth and Relationship Progression

### Goal
Increase realism quality with stronger persona control and structured relationship growth.

### Deliverables
- Character profile packs (life story, values, family/friends context)
- Relationship state machine (stranger/familiar/trusted)
- Topic-aware memory ranking and retrieval quality improvements

### Exit Criteria
- Character style drift reduced in multi-session conversation tests.
- Relationship progression is observable and consistent.

## V3: Time-Synced Social Simulation

### Goal
Add world dynamism: real-time rhythm, personal storyline evolution, and group-level behavior.

### Deliverables
- Real-world time synchronization (day/night, weekday/weekend behavior shifts)
- Personal storyline growth milestones
- Group social behavior and user intervention mechanics

### Exit Criteria
- Users can observe clear differences in character behavior by time context.
- Intervention actions have persistent impact on personal or group behavior arcs.
