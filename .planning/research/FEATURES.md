# Features Research

## Table Stakes (Must-have)

- Mobile-first loading and smooth crowd movement
- Tap moving character to start direct interaction
- Distinct character identity and speaking style
- Persistent memory per user-character pair
- Safe and bounded response behavior

## Differentiators

- Realistic social distance (stranger vs familiar pacing)
- Relationship progression over repeated meetings
- Contextual greetings linked to previous unfinished topics
- Character life simulation hooks (family, friends, routines) without breaking persona consistency

## Anti-Features (Not in v1)

- Full voice role-play and cinematic avatar systems
- Open multiplayer social platform with user-generated rooms
- Heavy simulation of full city systems before core loop validation

## Validation Signals

- Character click-through rate from crowd scene
- Session length in chat mode
- Return rate to same character within 7 days
- User-reported “this feels like a real person” sentiment
